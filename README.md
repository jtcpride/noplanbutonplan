# noplanbutonplan
NoPlan but OnPlanは、厳格な計画に囚われず、自発的な興味と優先順位に従ってプロジェクトを推進するために私が実践しているメソッドをまとめたものです。一見矛盾する「計画しない」と「進行中である」という概念を融合させ、興味駆動型で長期的な目標達成を目指します。締切やタスクが次々迫ってくるようなプロジェクトの管理には向きませんが、どんどん広がっていく興味とその実行のバランスを取るための仕組みってあまりないように思うので、使ってみてもらえたら嬉しいです。

0.GTDのカードの隅に穴を開けて、リングで綴じてください。
1.Note:あの本を読む、編み物をする、ゲームする、博物館行く、寝る、日記を書く、家庭菜園する・・・とにかくやりたいことをカード1枚に一つ片面に
2.Pick:ふと思った時でいいです。ペラペラめくってピンときたものを選びます。
3:only Practice:あとはやるだけ。

興味があってやりたいことの一番の問題は、忙しい日々、やらないといけないことの山の中で「そのやりたいことを忘れてしまう」ことです。
リングでちょっと綴じて、机の端にでも置いてるだけでふと目に止まった時にめくってみるだけで勝手にやりたいことは叶っていきます。

### Who is NoPlan but OnPlan for?

こんな方におすすめです：

- 多くのやりたいことや興味があること、やるべきことがあって、どれか一つに集中するのが難しい方
- 目標を設定しても、計画通りに進捗させることが苦手だと感じる方
- 従来の計画法にストレスを感じ、もっと柔軟なアプローチを求めている方
- 自身の特性を活かし、無理なく持続的に物事を進めたいと願う方
- 「進捗大丈夫です！」と胸を張って言える、自分らしい強さを持ちたい方
- 好きなことばかりやっていると焦りや自己嫌悪を感じてしまう方

### Name Origin

「NoPlan but OnPlan」という名称は、2005年ごろにThe North Faceの店舗で見かけたTシャツに書かれていた「No sight but On sight」（目が見えないか知らんけど一撃決めてやるぜ・・・私の中に響いたニュアンスでかなりの意訳）という言葉が涙が出るほど印象に残っていて、そこから着想を得ました。目が見えない、そりゃ普通は不利だろう。でも、On sight(クライミングで下見なしでいきなり上り切ってしまうこと、一撃）の世界では見えていないなんて関係ない。むしろ事前に見れないのはいつものこと、見えない分他の感覚は鋭く、なんなら見えないことは強みだ。計画を立てて物事を進めるのがどうしても苦手で、気が向いたもの自分の興味がある物事しかできない。でも、その弱みこそが実はやりたいことをやるには最適なんだ。という、弱みこそがそのまま強みなのだと肯定していきたい。そういう思いです。

- **NoPlan**: 厳密な事前計画に縛られない自由なアプローチを意味します。
- **OnPlan**: しかし、プロジェクトは「計画上」にあり、着実に進行している状態を意味します。

この名前は、色んな理由で計画に難しさを感じる人々が、独自のやり方で『進捗大丈夫です！』と胸を張れる、「制約を受け入れつつも、自らのペースで挑戦する」、そんな「計画なき計画の美学」を体現しています。

### Core Philosophy

1. **特性の肯定と活用**: 計画に難しさを感じる特性を「弱み」と捉えず、むしろ「創造的な進捗の源泉」として肯定的に捉え、活用します。

2. **興味駆動型のアプローチ**: 自分の興味や「やらねばならない」という内なる感覚に素直に従い、行動を選択します。これにより、モチベーションを維持しやすくなります。

3. **長期的な視点**: 短期的な計画の遅滞にいちいち落ち込まず、長期的な視点から見れば確実に進捗している状態を可視化します。

4. **心理的解放**: 完璧な計画を立てて実行することへのプレッシャーから解放され、柔軟かつ持続可能な方法で目標に近づきます。

### How It Works

NoPlan but OnPlanは、以下のシンプルなステップで実行されます。

1. **プロジェクトの集積**: 進めたいプロジェクトや興味のあることを、GTDのカード、システム手帳のリーフやNotionなどのデジタルツールに１トピック１ページとして書き出し、ストックしておきます。やりたいことがどんどん増えても大丈夫です。やりたいことができたらリーフもしくはページを追加していきます。

2. **レビューと選択**: 1日に一回でもどんな頻度でもいいです。ストックされたプロジェクトのリストをパラパラと見返し、「ピンときた」または「今やらねばならない」と感じたプロジェクトを選びます。「寝る（睡眠時間を確保する）」というプロジェクトにピンときたら、すぐに片付けて寝ることにします。

3. **タスクの実行**: 選んだプロジェクトから、何か一つ具体的なタスクを考え、すぐに実行します。その際、実行したことの概要と日時をリーフに書き加えます。

4. **進捗の積み重ね**: これを毎日繰り返すことで、個々の計画がないにも関わらず、全体としてプロジェクトが少しずつ、しかし確実に進捗していくのが感じられます。

このメソッドは、ミニ6穴のシステム手帳やNotionなど、柔軟な情報管理が可能なツールと相性が良いです。

### Value and Benefits

- **ADHD特性との適合**: 興味駆動型の行動パターンを活かし、計画の縛りによるストレスを軽減します。
- **現実的な進捗**: 完璧な計画実行への期待を手放し、小さな成功体験を積み重ねることで、長期的な成果に繋げます。
- **心理的安全性の確保**: 期待や義務からの解放により、心理的負荷が軽減され、前向きに取り組むことができます。
- **多様な生き方に対応**: 従来の計画法に苦手意識を持つ人々にとって、有効な選択肢となり得る新しいパラダイムを提供します。
